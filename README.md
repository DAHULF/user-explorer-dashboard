# User Explorer Dashboard

The learning project.

## About the project

This is a learning frontend project. It is a single-page dashboard for viewing, searching, and filtering users. I created it to practice HTML, CSS, JavaScript, DOM manipulation, and working with APIs.

## Features

## Features

- Responsive layout;
- User data loading from an external API;
- Search by name, email, company, role, or status;
- Filtering by status;
- Viewing active users only;
- Sorting by name, role, and date added;
- Pagination and selection of the number of cards per page;
- Loading, error, and empty states;
- Modular JavaScript structure;
- Randomly generated roles, statuses, and activities for demonstration purposes.

## Technologies

The project was built with:

- HTML5;
- CSS3;
- JavaScript;
- ES Modules;
- Fetch API;
- Lucide Icons.

While working on this project, i practiced the following skills:

## How to Run

To run the project locally:

1. Clone the repository;
2. Open the project folder in your code editor;
3. Open `index.html` using the Live Server extension for Visual Studio Code.
